# DuckTales

Небольшой двухмерный платформер по мотивам DuckTales, написанный на Python с использованием Pygame. Игрок управляет Скруджем Макдаком, собирает алмазы, сражается с противниками и проходит три уровня.

## Возможности

- три игровых уровня с отдельными текстовыми картами;
- меню выбора начального уровня;
- плавное горизонтальное движение и прыжки;
- атака тростью на земле и в воздухе;
- гориллы и ядовитые цветы;
- большие и маленькие алмазы;
- три единицы здоровья;
- подсчёт собранных денег;
- прокрутка уровня камерой;
- отдельный экран `GAME OVER`.

## Требования

- Python 3;
- Pygame;
- Pillow.

Все Python-зависимости перечислены в файле `requirements.txt`.

## Установка

Откройте терминал в корневой папке проекта и создайте виртуальное окружение:

```powershell
python -m venv .venv
```

Активируйте его в PowerShell:

```powershell
.venv\Scripts\Activate.ps1
```

Установите зависимости:

```powershell
python -m pip install -r requirements.txt
```

## Запуск

Запускайте игру из корневой папки проекта:

```powershell
python main.py
```

Это важно: изображения и карты загружаются по относительным путям из каталога `data`.

## Начало игры

1. На заставке нажмите любую клавишу или кнопку мыши.
2. На экране выбора уровня нажмите `1`, `2` или `3`. Также можно выбрать уровень левой кнопкой мыши.
3. Если выбран первый или второй уровень, после его прохождения игра продолжится на следующем уровне. Выбор третьего уровня сразу запускает последнюю карту.

## Управление

| Действие | Клавиши |
| --- | --- |
| Движение влево | `←` |
| Движение вправо | `→` |
| Прыжок | `Z` |
| Атака тростью | `Ctrl + X` |
| Выбор уровня | `1`, `2`, `3` или мышь |
| Закрытие экрана `GAME OVER` | любая клавиша или кнопка мыши |

Направление прыжка можно менять стрелками. Атака тростью работает как на земле, так и во время прыжка. Для атаки удерживайте левый или правый `Ctrl` и нажимайте `X`.

## Правила игры

У Скруджа есть три единицы здоровья. Столкновение с гориллой или ядовитым цветком отнимает одну единицу. После получения урона персонаж некоторое время проигрывает анимацию повреждения. Когда здоровье заканчивается, Скрудж падает за нижнюю границу экрана, после чего появляется экран `GAME OVER`.

Гориллу можно победить ударом трости. Если столкнуться с ней без активной атаки, Скрудж получит урон. Цветы уничтожить нельзя: их необходимо обходить или перепрыгивать.

На уровнях встречаются два вида алмазов:

- маленький алмаз приносит 1 000;
- большой алмаз приносит 5 000.

Текущее здоровье и количество денег отображаются в верхней части экрана. Для завершения уровня необходимо коснуться маркера выхода в конце карты.

## Структура проекта

```text
DuckTales/
├── main.py                 
├── images_creation.py      
├── requirements.txt       
├── README.md               
└── data/
    ├── first_level_map.txt
    ├── second_level_map.txt
    ├── third_level_map.txt
    └── ...   
```

`main.py` содержит основной игровой цикл, классы Скруджа и противников, обработку столкновений, камеру, меню и экраны состояния. `images_creation.py` вырезает отдельные изображения из исходных спрайт-листов с помощью Pillow.

## Формат карт

Уровни хранятся в обычных текстовых файлах внутри каталога `data`. Один символ соответствует одному объекту или одной клетке карты.

| Символ | Объект |
| --- | --- |
| `P` | начальная позиция игрока |
| `A` | горилла |
| `F` | ядовитый цветок |
| `d` | маленький алмаз |
| `D` | большой алмаз |
| `C` | выход с уровня |
| `G` | трава |
| `M` | средний слой земли |
| `E` | земля |
| `T` | ствол |
| `S` | пень |
| `R` | камень |
| `J` | камень с листьями |
| `L` | листья |
| `H` | шипы |
| `B` | кирпичи |
| `I` | статуя идола |
| `.` | пустая клетка |

При редактировании карты рекомендуется сохранять одинаковую ширину всех строк. На каждой карте должны присутствовать начальная позиция `P` и выход `C`.

## Возможные проблемы

### `ModuleNotFoundError: No module named 'pygame'`

Активируйте виртуальное окружение и повторно установите зависимости:

```powershell
python -m pip install -r requirements.txt
```

### Не найдены изображения или карты

Убедитесь, что команда `python main.py` выполняется из корневой папки проекта, а каталог `data` находится рядом с `main.py`.

### Не работает атака

Используйте сочетание `Ctrl + X`, а не только `X`. Поддерживается как левый, так и правый `Ctrl`.

### Скрипты PowerShell запрещены

Вместо активации окружения можно напрямую использовать его интерпретатор:

```powershell
.venv\Scripts\python.exe -m pip install -r requirements.txt
.venv\Scripts\python.exe main.py
```
